# Simmering and Huoviala (2023) — Large language models for ABSA

- **URL:** https://arxiv.org/abs/2310.18025
- **Authors:** Paul F. Simmering, Paavo Huoviala
- **Year:** 2023
- **Source:** arXiv API

## Key Findings

Compares zero-shot, few-shot, and fine-tuned LLMs for ABSA; highlights cost-performance and prompt-vs-fine-tune trade-offs.

## Relevance to This Project

Directly supports the thesis framing that robustness diagnostics should complement standard benchmark gains.
